# AI产品文档模板（工程版）- 主文档

## 文档说明

+ **定义和价值：** AI产品文档模板是产品经理用于驱动AI生成高保真产品原型（HTML）的标准化模板。它遵循"开发能看懂，AI可执行"的核心原则，旨在将产品方案沉淀为可被AI利用的知识资产，为公司未来的AI赋能与商业化奠定基础。
+ **标题规范：** **"产品母体"+"版本号"+"方案名称"+"AI文档"**。例如：《品牌商后台基础T6.11.0（操作日志方案）AI文档》。
+ **适用范围：** 新城扩建类项目，旧城改造项目有新的独立页面或模块的项目。
+ **工程版定义：** 一种"1+N"的"主-子"文档结构的需求范式，通过"主文档"定义全局上下文，通过"子文档"详述各独立模块。
+ **工程版适用场景：** 大、中版本。全新的复杂应用，对核心系统的重大重构，以及需要多人协作的大型项目。
+ **文档文件类型：** `.md`
+ **demo链接：** 放置于文档标题下方。

## 编写原则

+ **强调具体化：** 清晰表达（如"管理员想要按时间筛选操作日志"）、尽量量化（如"列表加载时间不超过2秒"）；
+ **强调因果性：** 如果...则...；因为...所以...;
+ **强调顺序性：** 开始...结束...; 从前...到后...; 1、2、3...;
+ **强调分段描述：** 尽量按具体功能闭环描述，如按某场景、模块、页面进行描述。
