安全专栏 · SECURITY

网关控制台首登强制改密，公网「admin/admin 裸奔」窗口关闭

控制台内置引导账号首次登录即强制改密；配合「破玻璃」恢复变量只认真值（1/true/yes/on）， 既保住了忘密码时的终极逃生通道，又不给误触发留缝。

优化

REFINE

CDS Mongo Console 整体重构：从自由 eval 式命令行改为「只读命令行 + 结构化写入面板」。 读操作保留命令行的灵活，写操作一律走表单化受控通道——从架构上让注入类攻击失去落点， 也让每一次写库都留下可核对的结构化记录。
