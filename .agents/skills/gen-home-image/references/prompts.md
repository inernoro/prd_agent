# 百宝箱图片提示词

## 浅色母版

Use case: stylized-concept. Create a light-theme MAP toolbox card artwork for **{agent name}**. Visual metaphor: **{responsibility metaphor}**. Use the supplied light MAP card as a material and composition reference, but create a distinct scene. Full-bleed warm ivory and pale stone architectural background must reach every edge and corner. Refined grayscale 3D objects, translucent glass, matte ceramic, soft warm daylight, restrained depth, sparse lime-green accents. Landscape composition, main subject slightly right of center, calm low-detail upper-left area, and all important objects inside the central 78% safe area. It must remain legible at small card size. No words, letters, numbers, logos, UI screenshots, watermarks, blue accents, rainbow colors, blank corners, borders, frames, vignette, or isolated sticker treatment.

## 深色配对

Use case: lighting-weather transformation for a dark-theme toolbox card. Use the first reference as the exact composition and the second reference as the MAP dark-theme material and lighting guide. Preserve camera, aspect ratio, object positions, scale, visual metaphor, full-bleed background, and upper-left quiet area. Change only lighting, palette, materials, shadows, and accent treatment. Use charcoal and graphite architecture reaching every edge, smoked glass, matte black ceramic, cool gray metal, soft directional studio light, restrained depth, and sparse lime-green accents. No words, letters, numbers, logos, watermarks, UI screenshots, blue accents, rainbow colors, blank corners, borders, frames, or vignette.

## 视觉验收

- 以 320x200 缩略图看时，职责隐喻仍可辨认。
- 四角没有纯白、透明、贴纸轮廓或突兀色块。
- 上部标题区没有高频细节；文字不与主体争抢。
- 明暗两张构图一致，不能像两个不同的智能体。
- 同一批次色温、镜头高度、材质精度和荧光绿用量相近。
