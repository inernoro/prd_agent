CDS Control Surface

以更克制的方式，掌控分支、构建与预览。

Cloud Dev Suite 将工作树管理、服务编排和预览路由收束到同一套控制界面， 保持稳定、清晰、可追踪，也让分支级环境在切换时仍然具备专业秩序感。

01 · Workspace Routing
工作树路由

统一收束分支、预览域名与服务入口，减少切换时的认知噪音。

02 · Build Orchestration
构建编排

在一个控制面里查看构建状态、热重启过程与多服务运行结果。

03 · Preview Proxy
预览代理

服务未完全就绪时平滑等待，避免白屏、闪断与无意义的 502 噪音。

04 · Runtime Audit
运行审计

关键日志、切换动作与环境状态保持连贯，方便定位每一次变化。
