trace运行详情或诊断区有 traceId，能关联 MAP session 与 CDS runtime
事件Timeline / eventslastEventSeq 增长，运行中能看到 SDK 事件同步
停止Stop 按钮和最终状态停止后最终状态为 stopped，后台不继续消耗
