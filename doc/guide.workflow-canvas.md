# 工作流画布 · 指南

> **类型**：guide | **版本**：v1.0 | **创建**：2026-03-15 | **状态**：已实现
>
> 画布是工作流的主编辑视图，支持可视化编排、实时执行、节点调试。本文档覆盖画布的全部操作。

---

## 1. 进入画布

路径：`/workflow-agent/:workflowId/canvas`

从工作流列表页点击某个工作流卡片的"画布"入口，或在编辑器页面切换到画布视图。页面加载时自动：

1. 拉取工作流数据（节点、边、变量）
2. 拉取最近一次执行记录（用于渲染执行状态）
3. **自动排列** — dagre 算法按 TB（从上到下）方向排列节点，等待 React Flow 测量真实节点尺寸后二次排列，确保布局整齐

---

## 2. 界面布局

```
┌──────────────────────────────────────────────────────────┐
│ TabBar：工作流名称 + [执行] [日志] [保存] [返回]         │
├──────────┬───────────────────────────────┬───────────────┤
│ 左侧     │ 中间                          │ 右侧 (可选)  │
│ 舱目录   │ React Flow 画布               │ 节点编辑面板  │
│ (可折叠) │ + 浮动工具栏                   │ / 执行日志   │
│          │ + MiniMap                      │              │
└──────────┴───────────────────────────────┴───────────────┘
```

### 2.1 顶部标题栏（TabBar）

| 按钮 | 功能 |
|------|------|
| **执行 / 重新执行** | 启动工作流执行；如有变量，弹出参数输入对话框 |
| **执行中 N/M + 取消** | 执行期间显示进度，可随时取消 |
| **日志 (N)** | 切换右侧执行日志面板 |
| **保存\*** | 保存当前画布修改（* 表示有未保存的变更） |
| **返回** | 返回工作流详情页 |

### 2.2 左侧舱目录

按类别（触发、处理、流程控制、输出）展示所有可用舱类型。**拖拽到画布**即可添加节点。

点击左上角 `+` 按钮可折叠/展开。

### 2.3 浮动工具栏（顶部居中）

| 按钮 | 快捷键 | 功能 |
|------|--------|------|
| `-` / `+` | — | 缩小 / 放大 |
| 百分比 | — | 当前缩放比例 |
| 适配画布 | — | 自动缩放让所有节点可见 |
| 100% | — | 重置缩放 |
| 自动排列 | `L` | dagre 重新排列所有节点 |
| 撤销 | `Ctrl+Z` | 撤销上一步操作 |
| 重做 | `Ctrl+Y` | 重做上一步操作 |
| 快捷键 | `?` | 显示快捷键提示面板 |

### 2.4 右侧节点编辑面板

点击画布上的节点即可打开。显示：

- 节点类型图标、名称（可编辑）
- 输入/输出插槽信息
- 配置字段表单（text / textarea / select）

### 2.5 右侧执行日志面板

通过标题栏"日志"按钮切换。显示：

- 实时执行日志（每 2.5s 轮询更新）
- 日志级别颜色（蓝=信息，绿=成功，红=错误，黄=警告）
- 节点名称标签、时间戳
- 自动滚动到最新条目
- 清空按钮

## 3. 节点操作

### 3.1 添加节点

| 方式 | 操作 |
|------|------|
| 从舱目录拖拽 | 从左侧面板拖拽到画布任意位置 |
| 连线拖放 | 从节点输出口拖线到空白区域，弹出节点选择器 |
| 边中点 "+" | 将鼠标悬停在连线上，点击 `+` 按钮插入中间节点 |

### 3.2 移动节点

直接拖拽节点主体即可移动位置。

### 3.3 删除节点/连线

- 选中后按 `Delete` 或 `Backspace`
- 或在标题栏点击"删除选中"（多选时出现）

### 3.4 多选

按住 `Shift` 键框选多个节点/连线。

### 3.5 编辑节点配置

点击节点打开右侧编辑面板，修改名称和配置字段。

### 3.6 断点

右键点击节点可切换断点（红点标记）。执行到断点节点时会暂停。

---

## 4. 连线操作

### 4.1 创建连线

从节点底部的**输出插槽**（绿色圆点）拖线到另一节点顶部的**输入插槽**。

- 禁止自连接（同一节点的输出连到自己的输入）
- 插槽支持 hover 放大高亮

### 4.2 删除连线

选中连线后按 `Delete`。

### 4.3 插入中间节点

鼠标悬停在连线中点，出现 `+` 按钮。点击后弹出节点选择器，选中的节点会自动插入到这条连线的中间，原边拆分为两条。

## 5. 执行工作流

### 5.1 启动执行

1. 点击标题栏 **执行** 按钮
2. 如果工作流定义了变量（如 API 地址、Cookie 等），弹出**执行参数对话框**
3. 填写必填变量后点击"执行"
4. 画布自动保存当前编辑内容，然后提交执行

### 5.2 执行过程可视化

执行期间，画布节点和连线实时更新状态：

| 元素 | 状态 | 视觉效果 |
|------|------|----------|
| **节点** | 运行中 | 蓝色脉冲边框 + 旋转加载图标 + 进度波浪条 |
| **节点** | 已完成 | 绿色边框 + 勾号 + 耗时 |
| **节点** | 失败 | 红色边框 + 警告图标 |
| **节点** | 断点暂停 | 琥珀色边框 + 暂停图标 |
| **连线** | 数据传输中 | 蓝色实线 + 粒子流动动画 |
| **连线** | 传输完成 | 绿色实线 + 脉冲波 |
| **连线** | 错误 | 红色虚线 + 闪烁 |

### 5.3 取消执行

执行期间标题栏显示 **"执行中 N/M"** 进度徽章和 **取消** 按钮。点击取消可中止执行。

### 5.4 执行日志

点击 **日志** 按钮打开右侧日志面板。日志包括：

- 节点级别的开始/完成/失败/跳过事件
- 总耗时统计
- 错误信息详情

## 6. 自动排列

### 6.1 手动触发

点击工具栏 **自动排列** 按钮（或按 `L` 键），dagre 算法会重新计算所有节点位置。

### 6.2 首次加载自动排列

画布加载时自动执行两轮排列：

1. **首轮**：使用默认尺寸（220x100）粗略排列
2. **二轮**：React Flow 测量节点真实尺寸后，用真实尺寸重新排列 + fitView

这确保了从模板导入或新建的工作流在打开时布局整齐。

## 8. 画布 vs 列表编辑器功能对比

画布和列表编辑器是同一工作流的两种视图。

| 功能 | 画布 | 列表编辑器 |
|------|:----:|:----------:|
| 添加/删除节点 | ✅ | ✅ |
| 编辑节点配置 | ✅ | ✅ |
| 创建/删除连线 | ✅ | — |
| 拖拽排列节点 | ✅ | — |
| 边中点插入节点 | ✅ | — |
| 自动排列布局 | ✅ | — |
| 撤销/重做 | ✅ | — |
| 执行工作流 | ✅ | ✅ |
| 取消执行 | ✅ | ✅ |
| 实时节点高亮 | ✅ | ✅ |
| 执行日志 | ✅ | ✅ |
| 变量输入 | ✅ | ✅ |
| 断点 | ✅ | ✅ |
| 单舱测试 | — | ✅ |
| AI 助手 | — | ✅ |
| cURL 导入 | — | ✅ |
| HTTP 配置面板 | — | ✅ |
| 节点回放 | — | ✅ |
| 产物预览 | — | ✅ |

> **总结**：画布侧重**可视化编排 + 结构操作 + 实时调试**，列表编辑器侧重**深度配置 + 单舱测试 + AI 辅助**。两者互补，建议用画布搭建拓扑结构、用编辑器精细配置。

## 9. 常见问题

### Q: 导入模板后节点位置混乱？

画布加载时会自动排列。如果仍不理想，按 `L` 手动触发自动排列。

### Q: 执行按钮变灰？

检查画布是否有至少一个节点。空画布不允许执行。

### Q: 执行后节点没有高亮？

确认后端 API 正常响应。画布通过 2.5s 间隔轮询获取执行状态，网络异常时高亮可能延迟。

### Q: 连线拖不到目标节点？

确认目标节点有输入插槽，且不是同一个节点（禁止自连接）。
