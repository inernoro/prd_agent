# 海鲜市场（配置市场）使用 · 指南

> **类型**：guide | **适用角色**：所有用户 | **最后更新**：2026-03-19
> **预计阅读**：5 分钟 | **难度**：入门

---

## 1. 功能概览

海鲜市场是平台的配置共享社区。浏览、搜索和 Fork 其他用户分享的优质配置。

| 配置类型 | 标签 | 来源应用 | 用途 |
|---------|------|---------|------|
| 提示词 | **「提示词」** | 文学创作 Agent | AI 对话和生成的提示词模板 |
| 参考图 | **「参考图」** | 视觉创作 Agent | 图像生成的风格参考配置 |
| 水印 | **「水印」** | 视觉创作 Agent | 图片水印样式配置 |

---

## 2. 操作路径

**入口方式**：
- 首页 → 点击「海鲜市场」快捷入口
- 导航栏 → 海鲜市场
- 直接访问 `/marketplace`
- 从 Agent 内跳转：`/marketplace?type=prompt`（直接筛选提示词）、`/marketplace?source=visual-agent`（筛选来自视觉创作的配置）

---

## 3. 完整操作流程

### 步骤 1：浏览市场

页面顶部：
- **搜索框**：「搜索配置名称...」
- **排序按钮**：**「热门」**（TrendingUp 图标）/ **「最新」**（Clock 图标）
- 来源标签：如 **「来自 视觉创作」**（通过 URL 参数自动筛选）

搜索框下方有 **类型筛选标签**：
- **「全部」** — 显示所有类型
- **「提示词」** — 仅显示提示词模板
- **「参考图」** — 仅显示参考图配置
- **「水印」** — 仅显示水印配置

### 步骤 2：预览配置

每个配置卡片显示：
- 配置名称
- 创建者
- Fork 次数（**「已被 Fork N 次」**）
- 配置描述
- 类型对应的预览渲染（提示词显示文本预览，参考图显示图片缩略图，水印显示效果预览）

### 步骤 3：Fork 使用

1. 找到满意的配置
2. 点击 **「Fork」**（GitFork 图标）
3. 成功后按钮变为 **「已添加」**（CheckCircle2 图标），弹出提示 **「已添加到我的配置」**
4. Fork 的配置自动出现在你的对应 Agent 的配置列表中

> **Pro Tip**：Fork 是完整复制，你可以在自己的空间里自由修改，不影响原作者的配置。

---

## 4. 如何发布配置到市场

1. 在对应 Agent 中找到你想分享的配置
2. 选择「发布到市场」
3. 填写标题、描述
4. 设置为公开
5. 发布后其他用户即可浏览和 Fork

---

## 5. 空状态处理

| 场景 | 显示内容 |
|------|---------|
| 市场无公开配置 | Store 图标 + **「暂无公开配置」** |
| 搜索无结果 | Store 图标 + **「没有找到匹配的配置」** |

---

## 6. 常见问题

**Q：Fork 别人的配置后，原作者能看到吗？**
A：原作者看不到谁 Fork 了，但配置卡片上的 Fork 次数会增加。

**Q：我发布的配置可以撤回吗？**
A：可以，在你的配置管理中将其设为非公开即可。

**Q：Fork 后原作者修改了配置，我的会同步更新吗？**
A：不会。Fork 是创建时的快照副本，互不影响。
