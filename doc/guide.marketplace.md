# 海鲜市场（配置市场）使用教程

> **类型**：guide | **适用角色**：所有用户 | **最后更新**：2026-03-19

---

## 1. 什么是海鲜市场

海鲜市场是平台的配置共享市场。用户可以浏览、搜索和 Fork 其他用户分享的优质配置，包括提示词模板、参考图配置和水印配置。

**适用场景**：
- 寻找高质量的提示词模板
- 浏览优秀的参考图配置
- 分享自己的创作配置给社区

---

## 2. 快速上手

### 步骤一：浏览市场

1. 进入海鲜市场页面
2. 通过分类筛选（提示词/参考图/水印）
3. 按热度或最新排序查看

### 步骤二：预览配置

1. 点击感兴趣的配置卡片
2. 查看详细预览和配置参数
3. 了解其他用户的 Fork 次数

### 步骤三：Fork 使用

1. 点击「Fork」按钮
2. 配置自动复制到你的工作区
3. 可在此基础上修改和使用

---

## 3. 配置类型

| 类型 | 说明 | 来源应用 |
|------|------|---------|
| 提示词 | AI 对话的提示词模板 | 文学创作 Agent |
| 参考图 | 图像生成的风格参考配置 | 视觉创作 Agent |
| 水印 | 图片水印样式配置 | 视觉创作 Agent |

---

## 4. 如何分享配置

1. 在对应 Agent 中找到你要分享的配置
2. 点击「发布到市场」
3. 填写标题、描述
4. 设置为公开后即可被其他用户看到

---

## 5. 常见问题

**Q：Fork 后可以修改吗？**
A：可以。Fork 是完整复制一份到你的空间，你可以自由修改。

**Q：我的配置被别人 Fork 了会怎样？**
A：不影响你的原始配置。Fork 次数会显示在你的配置卡片上。
