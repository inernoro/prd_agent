# 视频创作 Agent 使用教程

> **类型**：guide | **适用角色**：内容运营、自媒体创作者、市场营销 | **最后更新**：2026-03-19
> **预计阅读**：8 分钟 | **难度**：入门

---

## 1. 功能概览

视频创作 Agent 将文章自动转换为视频。三步流程：上传文章 → AI 生成分镜 → 编辑合成。

| 能力 | 说明 |
|------|------|
| 分镜生成 | AI 将文章拆分为多个场景，自动生成旁白和画面描述 |
| 场景编辑 | 逐个调整旁白台词和画面风格 |
| 预览渲染 | 每个场景独立渲染预览图 |
| 视频合成 | 一键合成带配音的完整视频 |
| 批量背景图 | 为所有场景批量生成背景图 |

---

## 2. 操作路径

**入口**：首页 → 点击「视频创作者」卡片 → 进入视频创作页面

顶部显示工作流进度条，三个阶段：
- **「上传文章」** → **「生成分镜」** → **「分镜编辑」**

---

## 3. 完整操作流程

### 步骤 1：上传文章

1. 点击 **「选择文件」**（Upload 图标）选择 Markdown 文件
2. 或直接在文本框中粘贴（提示：「粘贴 Markdown 文章内容，或点击上方按钮选择文件...」）
3. 可选填写：
   - **「视频标题」** — 不填则自动从文章提取
   - **「系统提示词」** — 控制旁白风格（如「旁白语言活泼轻松，面向初学者...」）
   - **「风格描述」** — 控制画面风格（如「科技感、深色背景、霓虹色系...」）
4. 顶部显示配置药丸：**「自动风格」**、**「风格」**、**「配置」**

> **Pro Tip**：风格描述对最终视频效果影响极大。建议明确指定色系和风格关键词，如「扁平插画、马卡龙色系、圆角元素」。

### 步骤 2：生成分镜

1. 点击 **「生成分镜标记」**（Sparkles 图标）
2. 按钮变为 **「生成中...」**，同时场景面板显示 **「生成中...」**（带旋转动画）
3. AI 将文章拆分为多个场景，每个场景自动生成：
   - 旁白台词
   - 画面描述
   - 场景类型标记

**8 种场景类型**：

| 类型 | 标签 | 用途 |
|------|------|------|
| intro | 开场 | 标题和主题引入 |
| concept | 概念 | 核心概念阐述 |
| steps | 步骤 | 操作流程展示 |
| code | 代码 | 技术代码展示 |
| comparison | 对比 | 方案对比 |
| diagram | 图表 | 数据和架构图 |
| summary | 总结 | 要点回顾 |
| outro | 结尾 | 致谢和引导关注 |

### 步骤 3：编辑分镜

每个场景卡片标题为 **「镜头 N」**，提供以下操作：

| 按钮 | 功能 |
|------|------|
| **「渲染分镜」** / **「重新渲染」**（Sparkles 图标） | 为该场景生成预览图 |
| **「背景图」**（Image 图标） | 生成场景背景图 |
| **「重试」**（RefreshCw 图标） | 重新生成该场景 |

- 每个场景有状态标记：**「等待」** → **「生成中」** → **「完成」** / **「失败」**
- 旁白台词可直接编辑（提示：「旁白台词（可编辑后右下角重新生成）」）

**批量操作**：
- 点击 **「生成背景图」** — 为所有场景批量生成背景
- 点击 **「渲染」** — 批量渲染所有场景预览

### 步骤 4：导出视频

1. 所有场景编辑完成后，点击 **「导出视频」**（Video 图标）
2. 等待渲染（状态从 **「排队」** → **「渲染中」** → **「完成」**）
3. 渲染完成后显示 **「视频已完成」**，可下载：
   - **「下载 MP4」** — 成品视频
   - **「SRT 字幕」** — 字幕文件
   - **「配音台词」** — 旁白文本
   - **「视频脚本」** — 完整分镜脚本

---

## 4. 调试功能

点击 **「调试」**（Bug 图标）打开调试面板 **「LLM 生成数据调试」**：
- **「场景概览」** — 各场景类型统计
- **「脚本 Markdown」** — 完整分镜脚本
- **「旁白文档」** — 所有旁白台词
- **「场景原始 JSON」** — 传给 Remotion 渲染的数据，点击 **「复制」** 导出

---

## 5. 历史任务

点击 **「历史任务」**（显示数量如 **「3 个」**）查看之前创建的视频任务，可以回看和重新下载。

---

## 6. 常见问题

**Q：视频渲染需要多久？**
A：取决于场景数量，通常 5-15 个场景需要 2-5 分钟。全程有状态提示（排队 → 渲染中 → 完成）。

**Q：可以只重新生成某个场景吗？**
A：可以。每个场景卡片独立操作，点击 **「重试」** 仅重新生成该场景。

**Q：旁白语言风格怎么控制？**
A：上传时在 **「系统提示词」** 中描述风格，如「专业严肃的技术讲解」或「轻松幽默的科普风格」。
