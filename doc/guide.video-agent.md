# 视频创作 Agent 使用教程

> **类型**：guide | **适用角色**：内容运营、自媒体创作者、市场营销 | **最后更新**：2026-03-19

---

## 1. 什么是视频创作 Agent

视频创作 Agent 可以将文章自动转换为视频。上传文章后，AI 自动生成分镜脚本、场景图片和配音，最终合成可下载的完整视频。

**适用场景**：
- 将技术博客转换为讲解视频
- 把产品介绍文章做成宣传短片
- 新闻资讯快速生成视频摘要

---

## 2. 快速上手

### 步骤一：上传文章

1. 进入视频创作 Agent 页面
2. 粘贴或输入文章内容（支持 Markdown）
3. 系统自动解析文章结构

### 步骤二：生成分镜

1. 点击「生成分镜」按钮
2. AI 将文章拆分为多个场景（开场、概念、步骤、对比、总结等）
3. 每个场景自动生成描述文案和画面提示

### 步骤三：编辑与预览

1. 可逐个编辑场景的文案和画面描述
2. 为每个场景生成预览图片
3. 不满意可单独重新生成某个场景

### 步骤四：合成视频

1. 确认所有场景后，点击「合成视频」
2. 等待渲染完成（实时显示进度）
3. 下载或在线播放成品视频

---

## 3. 场景类型

| 场景类型 | 用途 | 适用内容 |
|---------|------|---------|
| intro | 开场 | 标题、主题引入 |
| concept | 概念解释 | 核心概念阐述 |
| steps | 步骤展示 | 操作流程、教程 |
| code | 代码展示 | 技术内容 |
| comparison | 对比 | 方案对比、前后对比 |
| diagram | 图表 | 数据、架构图 |
| summary | 总结 | 要点回顾 |
| outro | 结尾 | 致谢、引导关注 |

---

## 4. 常见问题

**Q：视频渲染需要多长时间？**
A：取决于场景数量，通常 5-15 个场景的视频需要 2-5 分钟。

**Q：可以自定义背景音乐吗？**
A：目前使用系统预设配乐，后续将支持自定义上传。

**Q：输出视频的分辨率？**
A：默认 1080p，适合大多数社交媒体平台。
