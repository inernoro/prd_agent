# 视觉创作 Agent 使用教程

> **类型**：guide | **适用角色**：设计师、运营人员、内容创作者 | **最后更新**：2026-03-19

---

## 1. 什么是视觉创作 Agent

视觉创作 Agent 是一个专业级 AI 图像生成工作区。支持文生图、图生图、局部重绘、风格融合等功能，从文字描述到惊艳视觉作品，让创意触手可及。

**适用场景**：
- 快速生成产品概念图、宣传海报
- 基于参考图生成风格统一的系列图片
- 对已有图片进行局部修改和优化
- 多风格混合创造独特视觉效果

---

## 2. 快速上手

### 步骤一：进入工作区

1. 登录后在首页点击「视觉创作 Agent」
2. 创建新的工作区，或进入已有工作区

### 步骤二：文生图 — 你的第一张 AI 图片

1. 在对话框中输入图片描述，例如：「一只橘猫坐在窗台上，阳光洒落，水彩风格」
2. 选择图片尺寸（1:1、16:9、9:16 等）
3. 点击发送，等待 AI 生成图片
4. 图片生成后会显示在画板上，可以直接下载或继续编辑

### 步骤三：保存和管理

- 生成的图片自动保存在工作区中
- 可以对图片进行收藏、分组管理
- 支持批量导出

## 3. 核心功能详解

### 3.1 文生图

输入文字描述即可生成图片。

**描述技巧**：
- 主体 + 场景 + 风格：「戴草帽的女孩在花田中奔跑，宫崎骏风格」
- 加入光影描述：「黄金时刻，逆光，柔和散景」
- 指定画面构图：「俯视角度，对称构图，极简主义」

**参数说明**：
- **尺寸**：根据用途选择，社交媒体用 1:1，手机壁纸用 9:16
- **模型**：不同模型擅长不同风格，可在设置中切换

### 3.2 图生图

上传参考图，生成风格相似的新图片。

**使用方法**：
1. 在对话中上传参考图片
2. 描述你想要的变化：「保持构图不变，改为赛博朋克风格」
3. 调整相似度参数控制与原图的接近程度

### 3.3 局部重绘

精准编辑图片的特定区域，保持其余部分不变。

**使用方法**：
1. 在画板上选择一张已生成的图片
2. 使用画笔工具涂抹需要修改的区域
3. 输入修改描述：「将背景改为星空」
4. AI 仅重绘选中区域，自然融合到整体

### 3.4 风格融合

将多种艺术风格混合应用到同一张图片。

**使用方法**：
1. 上传多张不同风格的参考图
2. 描述融合方式：「用 A 的色彩搭配 + B 的笔触风格」
3. 调整各风格的权重比例

## 4. 画板操作

### 4.1 基本操作

- **缩放**：鼠标滚轮或双指缩放
- **平移**：按住空格拖拽
- **选择**：点击画板上的图片进行选中
- **删除**：选中后按 Delete 键

### 4.2 图片管理

- 画板上的图片支持拖拽排列
- 右键菜单可以快速操作（下载、收藏、重新生成等）
- 水印可在设置中配置

---

## 5. 进阶用法

### 5.1 参考图配置

在设置中预配置参考图，新建对话时自动应用，适合需要保持风格一致的批量创作。

### 5.2 提示词模板

将常用的描述保存为模板，提高创作效率。可在「配置市场」中浏览和 Fork 其他用户分享的优质提示词。

### 5.3 批量生成

输入多组描述，一次性生成多张图片，适合需要大量素材的场景。

---

## 6. 常见问题

**Q：生成的图片分辨率是多少？**
A：默认 1024×1024（1K），可在设置中选择更高分辨率。

**Q：生成一张图片需要多久？**
A：通常 5-15 秒，取决于所选模型和图片尺寸。

**Q：如何获得更好的生成效果？**
A：描述尽量详细具体，包含主体、场景、风格、光影等要素。可以参考配置市场中的高分提示词。

**Q：图片版权归谁？**
A：AI 生成的图片版权归创作者所有，可用于商业用途。
