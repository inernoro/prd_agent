# 视觉创作 Agent 使用教程

> **类型**：guide | **适用角色**：设计师、运营人员、内容创作者 | **最后更新**：2026-03-19
> **预计阅读**：10 分钟 | **难度**：入门

---

## 1. 功能概览

视觉创作 Agent 是专业级 AI 图像生成工作区，提供画板 + 对话双面板界面。

| 能力 | 说明 |
|------|------|
| 文生图 | 输入文字描述生成高质量图片 |
| 图生图 | 上传参考图，生成风格相似的新图片 |
| 局部重绘 | 在画板上涂抹区域，仅重绘选中部分 |
| 风格融合 | 多种艺术风格混合应用 |
| 水印配置 | 自定义水印样式并自动应用 |
| 参考图管理 | 预配置参考图，批量保持风格一致 |

---

## 2. 操作路径

**入口**：首页 → 点击「视觉设计师」卡片 → 进入全屏工作区

页面布局（三栏）：
- **左侧**：工具栏（光标、选择等工具）
- **中央**：画板区域（生成的图片展示在这里）
- **右侧**：对话面板，顶部显示「Hi，我是你的 AI 设计师」

---

## 3. 完整操作流程

### 步骤 1：创建工作区

1. 进入视觉创作 Agent，看到工作区列表
2. 点击 **「创建」** 新建工作区
3. 输入工作区名称（如「产品海报设计」）
4. 进入全屏编辑界面

### 步骤 2：文生图 — 你的第一张 AI 图片

1. 在右侧对话面板底部输入框中输入描述
2. 选择图片参数：
   - **尺寸**：点击参数标签选择（如 `1K·1:1`、`1K·16:9`）
   - **模型**：在参数标签中选择生成模型
3. 点击发送按钮
4. 图片生成后自动显示在中央画板上

**高质量描述模板**：

```
主体 + 场景 + 风格 + 光影 + 构图

示例：
「戴草帽的女孩在向日葵花田中奔跑，宫崎骏动画风格，
  黄金时刻暖色调，逆光剪影，广角镜头」

「极简主义办公桌面，白色背景，一台银色笔记本电脑，
  一杯冒着热气的咖啡，俯视角度，产品摄影风格」
```

> **Pro Tip**：描述要具体。「一只猫」不如「一只蓝色眼睛的银渐层英短猫趴在窗台上，窗外是雨天，油画风格」。

### 步骤 3：图生图 — 基于参考图生成

1. 在对话中点击上传图片按钮
2. 上传参考图
3. 输入描述说明你想要的变化：「保持构图和配色，改为水彩风格」
4. 生成结果出现在画板上

### 步骤 4：局部重绘

1. 在画板上 **点击** 已生成的图片将其选中
2. 使用画笔工具涂抹需要修改的区域
3. 在对话框中输入修改描述：「将背景改为星空」
4. AI 仅重绘涂抹区域，边缘自然融合

### 步骤 5：对话式迭代

在右侧对话面板中可以持续对话迭代：
- 「站起来」— 修改角色姿态
- 「换个背景」— 更换场景
- 「更鲜艳一些」— 调整色彩

对话面板中每条消息旁有 **「重试」** 按钮，不满意可重新生成。

## 4. 画板操作速查

| 操作 | 方式 |
|------|------|
| 缩放画布 | 鼠标滚轮 / 顶部工具栏百分比选择 |
| 适配画布 | 点击顶部工具栏 **「适配」** |
| 平移画布 | 按住空格 + 鼠标拖拽 |
| 选中图片 | 点击画板上的图片 |
| 下载图片 | 选中图片 → 右键菜单 → 下载 |

## 5. 进阶配置

### 参考图配置

在 **「配置」** 中预设参考图，新建对话时自动作为风格参考。适合品牌统一风格的批量创作。

### 水印配置

在设置中配置水印样式（文字、位置、透明度），生成的图片右下角可见水印标记（如画板上的「米多AI生成」）。

### 提示词模板

常用描述可保存为模板。到 **海鲜市场**（`/marketplace?type=prompt`）浏览其他用户分享的高质量提示词，一键 **「Fork」** 使用。

## 6. 常见问题

**Q：生成一张图需要多久？**
A：通常 5-15 秒。生成过程中对话面板会实时显示进度，不会出现空白等待。

**Q：图片分辨率多少？**
A：参数标签中显示，如 `1K·1:1` 表示 1024×1024。

**Q：如何保持系列图片风格一致？**
A：使用参考图配置 + 统一的提示词模板。最佳实践：先生成一张满意的图，将其设为参考图，后续生成自动参考该风格。

**Q：图片版权归谁？**
A：AI 生成的图片归创作者使用，可用于商业用途。
