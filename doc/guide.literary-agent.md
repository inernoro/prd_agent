# 文学创作 Agent 使用教程

> **类型**：guide | **适用角色**：内容编辑、自媒体运营、作家 | **最后更新**：2026-03-19

---

## 1. 什么是文学创作 Agent

文学创作 Agent 是智能文章配图与文学润色助手。一键为文章生成与内容完美契合的插画，支持水彩、油画、素描等 20+ 艺术风格，同时提供 AI 辅助改写功能，让每篇文章都成为视觉盛宴。

**适用场景**：
- 公众号文章需要配图，但没有设计资源
- 小说章节需要插画来增强阅读体验
- 文章需要润色优化，提升表达力
- 系列内容需要统一风格的批量配图

---

## 2. 快速上手

### 步骤一：创建作品

1. 登录后进入文学创作 Agent
2. 点击「新建作品」
3. 粘贴或输入你的文章内容

### 步骤二：一键配图

1. 文章导入后，AI 自动分析内容语义
2. 点击「生成配图」，选择喜欢的艺术风格
3. AI 为文章关键段落自动生成配图
4. 可对单张配图进行微调或重新生成

### 步骤三：导出成品

- 导出带配图的完整文章
- 支持单独下载配图
- 可直接复制到其他编辑器

---

## 3. 核心功能详解

### 3.1 智能配图

AI 分析文章内容，自动为关键段落生成契合的插画。

**工作原理**：
1. AI 提取文章中的场景、情感、意象
2. 将文字信息转化为图像描述
3. 根据选择的风格生成配图
4. 自动适配文章排版位置

**风格选择建议**：
| 风格 | 适用场景 |
|------|---------|
| 水彩 | 散文、游记、情感类文章 |
| 油画 | 历史题材、厚重叙事 |
| 素描 | 技术文章、极简风格 |
| 国风 | 古诗词、传统文化内容 |
| 动漫 | 轻小说、青春故事 |

### 3.2 风格迁移

将已有图片转换为指定的艺术风格。

**使用方法**：
1. 上传原始图片
2. 选择目标风格或上传风格参考图
3. 调整迁移强度（轻微/中等/完全）
4. 生成风格迁移后的图片

### 3.3 文学润色

AI 辅助改写文章，提升文字表达力。

**润色模式**：
- **精简模式**：删减冗余，提炼要点
- **扩展模式**：丰富细节，增加描写
- **风格转换**：正式 ↔ 轻松、现代 ↔ 古典

**使用方法**：
1. 选中需要润色的段落
2. 选择润色模式
3. AI 给出改写建议，可选择接受或忽略

### 3.4 批量处理

一键为整篇文章生成系列配图。

**使用方法**：
1. 导入完整文章
2. 选择统一的艺术风格
3. 设置配图密度（每隔几段配一图）
4. 一键生成所有配图
5. 逐张审核，不满意的单独重新生成

## 4. 进阶用法

### 4.1 风格模板

将常用的风格参数保存为模板，系列文章可以保持风格一致性。

### 4.2 提示词优化

AI 自动生成的图像描述可以手动调整，让配图更贴合你的想象。

### 4.3 配置市场

浏览其他创作者分享的风格配置和提示词模板，一键 Fork 使用。

## 5. 常见问题

**Q：文章字数有限制吗？**
A：单篇文章建议不超过 5 万字。超长文章建议分章节处理。

**Q：配图和文章内容不匹配怎么办？**
A：可以手动编辑 AI 生成的图像描述，使其更贴合你的意图，然后重新生成。

**Q：可以用自己的图片作为风格参考吗？**
A：可以。上传参考图后，AI 会学习图片的风格特征并应用到配图生成中。

**Q：支持哪些语言的文章？**
A：目前支持中文和英文文章的内容分析与配图生成。
