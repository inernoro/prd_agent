# 文学创作 Agent 使用教程

> **类型**：guide | **适用角色**：内容编辑、自媒体运营、作家 | **最后更新**：2026-03-19
> **预计阅读**：8 分钟 | **难度**：入门

---

## 1. 功能概览

文学创作 Agent 将文章和插画结合——导入文章后，AI 自动在关键段落生成风格统一的配图。

| 能力 | 说明 |
|------|------|
| 智能配图 | AI 提取文章语义，自动为关键段落生成插画 |
| 风格迁移 | 水彩、油画、素描等 20+ 艺术风格 |
| 文学润色 | AI 辅助改写段落，提升表达力 |
| 批量处理 | 一键为全文生成系列配图 |
| 提示词模板 | 保存/复用常用配图风格 |

---

## 2. 操作路径

**入口**：首页 → 点击「文学创作者」卡片 → 进入作品列表

---

## 3. 完整操作流程

### 步骤 1：创建文章工作区

1. 在作品列表页，点击 **「新建文章」**（Plus 图标）
2. 弹出对话框「请输入文章标题」，输入标题（默认「未命名」）
3. 点击 **「创建」** 进入编辑器

> **Pro Tip**：可以先用 **「新建文件夹」**（FolderPlus 图标）按系列分组管理文章，如「小红书系列」「技术博客」。

### 步骤 2：导入文章内容

1. 在编辑器中，上传阶段显示 **「选择文件」** 按钮
2. 支持 `.md`、`.txt`、`.markdown` 格式
3. 也可以直接在文本框中粘贴内容（提示：「粘贴 Markdown 文章内容，或点击上方按钮选择文件...」）

### 步骤 3：生成分镜配图

1. 文章导入后，点击主操作按钮开始生成
2. AI 进入思考阶段，页面显示：
   - **「AI 正在思考...」** — 分析文章结构
   - **「AI 正在输出分镜脚本...」** — 生成配图方案
3. 生成完成后，每个配图位置显示为 **「配图 N」** 卡片

> 等待过程中可以看到 **「思考过程」** 和 **「输出内容」** 两个区域的实时内容，不会出现空白等待。

### 步骤 4：编辑和微调配图

每张配图卡片提供以下操作：

| 按钮 | 功能 |
|------|------|
| **「重新生成」**（RefreshCw 图标） | 对这张配图重新生成 |
| **「预览」**（Eye 图标） | 大图预览 |
| **「删除」**（Trash2 图标） | 删除这个配图位置 |

- 点击配图下方文案区域（提示：「点击修改配图文案...」）可手动调整 AI 生成的图像描述
- 修改文案后点击重新生成，配图会按新描述更新

### 步骤 5：配置风格

1. 点击 **「配置」**（Settings 图标）打开配置面板
2. 可设置：
   - 水印配置
   - 生成模型选择
   - 参考图管理
3. 从 **「选择我的提示词模板」** 浏览已有模板
   - 搜索框：「搜索提示词...」
   - 排序：**「热门」** / **「最新」**
   - 点击 **「使用」** 应用模板

### 步骤 6：导出

- 点击 **「发布文章」**（Wand2 图标）发布带配图的文章
- 点击 **「下载」**（Download 图标）导出文章和配图

## 4. 文件夹管理

| 操作 | 方法 |
|------|------|
| 新建文件夹 | 列表页点击 **「新建文件夹」**，输入名称（默认「我的文件夹」），点击 **「创建」** |
| 重命名文件夹 | 右键文件夹 → **「重命名」** → 输入新名称 → 点击 **「重命名」** |
| 删除文件夹 | 右键文件夹 → **「删除」** → 确认后文件夹内文章移至「未分类」 |
| 重命名文章 | 右键文章 → **「重命名」** → 输入新标题 → 点击 **「确定」** |
| 删除文章 | 右键文章 → **「删除」** → 确认（此操作无法撤销） |

## 5. 风格选择建议

| 风格 | 适用场景 | 效果 |
|------|---------|------|
| 水彩 | 散文、游记、情感文章 | 柔和通透，文艺感强 |
| 油画 | 历史题材、厚重叙事 | 质感浓郁，有年代感 |
| 素描 | 技术文章、极简风格 | 线条清晰，干净利落 |
| 国风 | 古诗词、传统文化 | 水墨意境，古典韵味 |
| 动漫 | 轻小说、青春故事 | 明快色彩，年轻活泼 |

## 6. 常见问题

**Q：生成的配图和文章内容不匹配？**
A：点击配图文案区域手动修改 AI 生成的图像描述，使其更贴合你的意图，然后点击 **「重新生成」**。

**Q：如何保持系列文章配图风格一致？**
A：在配置中设置统一的参考图和提示词模板。或到海鲜市场 Fork 一套风格模板反复使用。

**Q：文章字数有限制吗？**
A：单篇建议不超过 5 万字。超长文章建议分章节创建多个工作区。
