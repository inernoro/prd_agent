使用路径

单次运行复盘：在 CDS Agent 面板完成只读巡检，导出 trace bundle，用 traceId、lastEventSeq、artifact 和 logs 复盘。

多运行时治理：打开 Adapter matrix，查看默认 claude-agent-sdk 可路由状态和 planned adapter 缺失 contract。

团队治理：打开权限/组织治理，查看 scope 和 Repository/Profile/Approval owner 卡片，并按 path 跳转处理。

定时巡检：通过 workflow schedule 触发 CdsAgentRun，知识库治理只用 kb_list/kb_search/kb_read。
