# 移动端视觉检查矩阵 · 规则

> **版本**：v1.0 | **日期**：2026-07-02 | **状态**：已落地

## 目的

本规则用于把移动端视觉检查从“随手截几张图”改成可复用的页面状态矩阵。检查对象不是路由数量，而是用户会遇到的页面状态：列表、详情、编辑、画布、弹层、空状态、加载状态、分享态都要分开算。

本文件只定义检查范围和修正口径；实际截图、录屏、manifest、验收报告仍按 `create-visual-test-to-kb` 技能进入 CDS 验收中心，禁止把验收产物提交到 `doc/`。

## 分级口径

| 级别 | 含义 | 处理顺序 | 通过门槛 |
|------|------|----------|----------|
| P0 | 移动端核心路径，用户高频使用或本次 UI 改造直接影响 | 每轮移动端改造必测 | 页面可进入、主内容可读、无横向文档溢出、顶部和底部不遮挡 |
| P1 | 登录后常用工作页，共享顶部栏、TabBar、Surface、滚动容器影响面 | 共享组件改动后回归 | 主要工作流可达，工具栏不换行，滚动归属清晰 |
| P2 | 公开页、深链页、低频后台页、模板页 | 发布前或专项回归 | 不白屏，不出现桌面布局硬塞手机，关键 CTA 可触达 |

## 通用修正原则

| 维度 | 检查点 | 修正口径 |
|------|--------|----------|
| 顶部栏 | App 名称、返回、通知、设置等挤压换行 | 移动端顶部只保留一个主标题和最多两个主操作；次级操作进更多菜单；标题单行截断 |
| 顶部可读性 | 顶部 Tab/筛选悬浮在列表上方、背后文字透出 | 导航可以有毛玻璃，但必须有稳定实底；暗色背景不低于 90% 不透明感，浅色背景不低于 92% 不透明感；禁止靠背后内容提供对比 |
| 主按钮 | 同一工具栏里主操作被横向遮挡、被更多菜单吞掉或排在次级操作后 | 每个页面同一层级只保留一个主按钮，优先直接展示；移动端主按钮可用底部 FAB 或右侧固定按钮，不能放在横滑末端 |
| 图标按钮 | 同组图标大小不一、触控框不一、视觉基线不齐 | 同组 icon-only 操作统一触控框和图标尺寸；紧凑工具栏最小 32px 方形触控框，普通工具栏优先 36-44px |
| 主面积 | 顶部、筛选、Tab 占用过高 | 首屏优先给内容；筛选区合并为一行可横滑控件或折叠控件 |
| 滚动 | 页面不能滑、横向卡住、body 出现横向溢出 | 只有最近内容层滚动；根容器 `min-width:0`；横向滚动必须限制在局部 lane |
| Surface | 单独看好看，组合起来像旧前端页面 | 减少嵌套卡片；统一半透明、边框、阴影和间距；卡片只用于独立对象 |
| 毛玻璃 | 对话和浮层过透明导致文字不可读 | 背景遮罩不低于可读阈值；正文容器使用稳定背景和 blur，不依赖背后内容对比 |
| 系统分享 | 分享只支持站内团队，无法分享到系统其他应用 | 分享入口必须优先支持系统分享能力；不支持时回退复制链接；站内团队分享只能作为其中一个选项 |
| 目录导航 | 详情页用“返回目录/返回列表”强迫用户离开正文 | 移动端目录、侧栏、筛选树统一用左侧抽屉或底部 sheet；浏览器/系统返回负责退出当前页面，不把返回按钮当作主导航 |
| 时间线/网格 | 左侧固定列 + 右侧最小卡宽导致手机重叠 | 手机端时间线一律单列堆叠；日期/分组标题在卡片上方，禁止保留桌面固定左列 |
| 底部导航 | 遮挡内容、调试条覆盖、中心按钮挤压 | 内容区预留 safe-area 和底部导航高度；调试状态不得覆盖生产主路径 |
| 证据 | 截图看不出在验证什么 | 每张指向性截图必须框出问题或通过点，并写明“这张证明什么” |

## 每轮检查输出

每次视觉检查至少输出下面四张表，缺一项不算完整：

| 表 | 内容 | 用途 |
|----|------|------|
| 页面状态覆盖表 | 本文件矩阵中的状态、是否覆盖、截图编号 | 防漏页 |
| 问题修正表 | 问题现象、严重级、共性根因、涉及组件、修正方案 | 防头痛医头 |
| 共享组件影响表 | HeadBar、TabBar、Surface、滚动容器、底部导航的使用页 | 一处改动覆盖多页 |
| 证据回读表 | 截图是否加载完成、是否入镜、是否有框选、是否能独立证明结论 | 防截图无效 |

## 判定口径

| 判定 | 条件 |
|------|------|
| 通过 | P0 全部通过，P1 无 P0/P1 级问题，P2 无白屏或不可进入 |
| 有条件通过 | P0 无阻断，P1/P2 存在少量 P2/P3 视觉债且列入修正表 |
| 不通过 | 任一 P0 页面白屏、不可滚动、主操作不可触达、横向文档溢出、顶部或底部遮挡核心内容 |
