## 1. 你会用到的页面

进入周报 Agent 后，顶部固定 3 个页签：

| 页签 | 说明 |
|------|------|
| 周报 | 写周报、看周报、打卡、AI生成草稿 |
| 团队 | 团队周报列表、团队成员、团队周报AI分析 |
| 设置 | 个人设置 + 管理设置 |

设置页现在是以下模块：

- 个人设置：`我的数据源`、`AI生成周报Prompt`
- 管理设置：`模板管理`、`团队管理`、`团队周报AI分析Prompt`

![设置页模块总览（示意）](images/report-agent/settings-overview.png)

## 2. 5分钟快速上手（普通成员）

### 步骤 1：先记当天工作

1. 打开 `周报` 页签。
2. 进入“日常记录/打卡”区域。
3. 输入你今天做的事情，至少保存 1~3 条。

![日常记录示意](images/report-agent/daily-log-quickstart.png)

### 步骤 2：创建周报草稿

1. 点击“创建周报”。
2. 选择团队和模板。
3. 选择一种方式：
   - `手动填写`
   - `AI生成周报草稿`

![创建周报方式选择（示意）](images/report-agent/create-report-mode.png)

### 步骤 3：提交

1. 检查每个板块内容。
2. 必要时点击 `AI重新生成草稿`（会覆盖当前草稿）。
3. 点击提交，等待负责人审阅。

## 3. 管理员首次搭建（建议一次做完）

> 建议顺序：先权限，再团队，再模板。这样普通成员一进来就能用。

### 步骤 1：分配权限

给不同角色分配权限（至少）：

- 所有使用者：`report-agent.use`
- 管理员：`report-agent.team.manage`、`report-agent.template.manage`
- 全局查看者（可选）：`report-agent.view.all`

### 步骤 2：创建团队

路径：`设置 -> 团队管理`

1. 新建团队。
2. 选择负责人。
3. 设置可见性（团队成员互看 / 仅负责人可看）。
4. 保存。

### 步骤 3：添加成员

1. 选择团队。
2. 点击“添加成员”。
3. 设置角色（成员/副负责人）。
4. 保存。

### 步骤 4：准备模板

路径：`设置 -> 模板管理`

1. 新建模板或使用现有模板。
2. 检查板块是否覆盖你团队周报习惯。
3. 建议至少包含：本周完成、风险阻塞、下周计划。

## 4. 普通成员日常使用

## 4.1 写周报（手动）

1. 创建周报后进入编辑页。
2. 按板块逐条填写。
3. 点“保存”可保留草稿，不会提交。

## 4.2 写周报（AI）

1. 创建时选 `AI生成周报草稿`，或进入草稿后点 `AI重新生成草稿`。
2. 系统会根据：
   - 当周日常记录
   - MAP平台工作记录（如果你没关闭）
   - 当前生效的个人 Prompt
   生成内容。
3. 生成后会显示本次模型信息（兜底时会标注“规则兜底”）。

## 4.3 日常记录标签

标签管理已经放在“日常记录”页，不在设置页：

1. 在日常记录区点击“管理标签”。
2. 支持新增、修改、删除自定义标签。
3. 提交记录时支持多选标签，且至少要选一个。
4. “其它”固定在最后显示。

## 4.4 评论与点赞

1. 进入周报详情。
2. 每个板块都可就地评论（输入框在当前板块内，不在页面底部）。
3. 点赞可再次点击取消。

## 5. 团队负责人专用操作

## 5.1 团队周报列表

路径：`团队` 页签，默认进入“团队周报列表”。

你可以：

1. 切换“我管理的团队”。
2. 查看成员提交状态。
3. 进入某成员周报做审阅或退回。

## 5.2 团队周报AI分析

1. 在“我管理的团队”视角点击 `团队周报AI分析`。
2. 点击“生成汇总”或“重新生成”。
3. 得到团队维度的结构化分析内容。

注意：

- 在“我加入的团队”视角，成员不能操作团队 AI 分析入口。

## 5.3 成员与权限边界

- 负责人/副负责人：可生成团队分析、管理成员。
- 普通成员：看自己权限范围内内容，不可管理团队。

## 6. Prompt 配置（个人 + 团队）

## 6.1 个人：AI生成周报Prompt

路径：`设置 -> AI生成周报Prompt`

1. 查看系统默认 Prompt（只读）。
2. 填写自定义 Prompt。
3. 点击“保存自定义 Prompt”。
4. 需要回退时点击“恢复系统默认”。

![个人Prompt配置（示意）](images/report-agent/personal-prompt-settings.png)

## 6.2 团队：团队周报AI分析Prompt

路径：`设置 -> 团队周报AI分析Prompt`

1. 先选择团队。
2. 查看系统默认 Prompt（只读）。
3. 填写团队自定义 Prompt。
4. 点击“保存团队自定义 Prompt”。
5. 如需回退，点击“恢复系统默认”。

这个配置会直接影响“团队周报AI分析”的生成风格和输出重点。

![团队Prompt配置（示意）](images/report-agent/team-summary-prompt-settings.png)

## 7. 常见问题（FAQ）

### Q1：为什么我看不到“团队周报AI分析”按钮？

A：你当前可能在“我加入的团队”视角，或不是负责人/副负责人。

### Q2：为什么 AI 草稿内容很少？

A：先检查三件事：
1. 当周日常记录是否足够；
2. MAP 数据源是否被关闭；
3. Prompt 是否写得太窄。

### Q3：为什么“添加扩展源”不能填表单？

A：该入口当前是“精细打磨中”提示态，暂不开放实际接入界面。

### Q4：我误点了 AI 重新生成，原文没了怎么办？

A：AI 重新生成会覆盖当前草稿。建议先保存关键内容后再生成。

### Q5：团队 Prompt 和个人 Prompt 会冲突吗？

A：不会。  
个人 Prompt 作用于“个人周报草稿生成”；  
团队 Prompt 作用于“团队周报AI分析”。
