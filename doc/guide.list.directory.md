# MAP 平台文档索引 · 指南

> **版本**：v2.1 | **日期**：2026-07-18 | **状态**：已落地

本目录由 `doc/index.yml` 的同一份文件清单生成。文档标题以各文件 H1 为准；新增、重命名或删除文档后必须重新同步两份索引。

## 使用方式

- 产品和研发按下列七类浏览长期文档。
- 外部同步工具读取 `doc/index.yml`。
- 标有 `grandfather 保留` 的非周报报告只因存量契约引用暂留，解除引用后回收。
