# 周报管理 Agent 使用教程

> **类型**：guide | **适用角色**：团队负责人、项目经理、全体成员 | **最后更新**：2026-03-19

---

## 1. 什么是周报管理 Agent

周报管理 Agent 提供团队周报的创建、管理和 AI 辅助生成功能。支持日志记录、数据源集成、团队协作，让周报从「苦差事」变成「一键完成」。

**适用场景**：
- 每周需要汇总团队工作进展
- 希望从 Git 提交记录自动生成周报
- 需要团队成员协作编写周报

---

## 2. 快速上手

### 步骤一：创建团队

1. 进入周报管理 Agent
2. 在「团队」标签页创建团队
3. 邀请团队成员并设置角色（负责人/成员）

### 步骤二：配置数据源

1. 进入「设置」标签页
2. 配置数据源（如 Git 仓库、项目管理工具）
3. 设置周报模板（可选）

### 步骤三：生成周报

1. 在「周报」标签页点击创建新周报
2. AI 自动从数据源拉取本周记录
3. 点击「AI 生成」，自动汇总为结构化周报
4. 审核并提交

---

## 3. 核心功能

### 3.1 日志记录

每日记录工作内容，周报生成时自动汇总。

### 3.2 AI 生成

AI 分析你的日志和 Git 提交，自动生成结构化周报，包含：
- 本周完成项
- 进行中的任务
- 下周计划
- 风险与阻塞

### 3.3 团队汇总

团队负责人可查看所有成员的周报，一键生成团队汇总报告。

### 3.4 评论协作

团队成员可以在周报上添加评论，异步沟通工作进展。

---

## 4. 常见问题

**Q：数据源支持哪些平台？**
A：目前支持 Git 仓库提交记录。后续将支持更多项目管理工具。

**Q：周报可以导出吗？**
A：支持导出为 Markdown 格式。
